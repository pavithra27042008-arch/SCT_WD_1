Responsive Landing Page

📌 Project Overview

This project is a responsive landing page developed using **HTML, CSS, and JavaScript**. It features a fixed navigation bar that changes its background color when the page is scrolled and includes hover effects on menu items. The website is fully responsive and works across desktop, tablet, and mobile devices.

✨ Features
- Responsive Design
- Fixed Navigation Bar
- Navigation Color Change on Scroll
- Hover Effects on Navigation Menu
- Hero Section with Call-to-Action Button
- Smooth User Experience
- Clean and Modern UI

🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript
